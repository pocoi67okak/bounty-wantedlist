# Баунти

Spigot-плагин под Minecraft `1.21.8` для команды `/wantedlist`.

## Сборка

Jar собирается через GitHub Actions. После пуша открой вкладку **Actions**, выбери последний запуск **Build** и скачай artifact **Bounty**.

Локальная сборка не требуется.

## Команды

- `/wantedlist` - открыть главное меню.
- `/wantedlist decline` - отказаться от взятой ставки.

## Логика

- В главном меню есть кнопки: список розыска, поставить розыск, мои ставки.
- Ставка создается через чатовый ввод ника и меню, куда кладутся ресурсы.
- Ресурсы можно забрать из меню "Мои ставки", пока ставку никто не взял.
- Один игрок может взять только одну ставку.
- После убийства цели награда выдается убийце, а создателю ставки приходит уведомление.
